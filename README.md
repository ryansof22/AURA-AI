# ✨ AURA: Strategic Personal AI Assistant

AURA is an initiative AI designed to be a high-level personal assistant, focusing on data strategy, learning support, and proactive engagement.

## 🚀 Key Features
- **Hybrid Memory System**: Combines short-term chat context (last 100 rows) and long-term personal guidelines using Google Sheets.
- **Strategic Insights**: Leverages 'Think with Google' and 'Google Research' perspectives for business and tech discussions.
- **Language Support**: Expert assistance for Japanese learning (JLPT N5 focus) including translations and Romaji.
- **Proactive Initiative**: Greets and reminds the user based on specific schedules and habits.

## 🛠️ Tech Stack
- **Core AI**: Gemini 2.5 Flash API
- **Web Framework**: Streamlit
- **Database**: Google Sheets API (via Gspread)
- **Language**: Python 3.x
